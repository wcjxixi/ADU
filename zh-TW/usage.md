# 使用

## 按鈕使用方法 <a href="#button-usage" id="button-usage"></a>

### 一般界面 <a href="#general" id="general"></a>

* **點選**：切換主螢幕的下一個資訊界面
* **雙擊**：跳轉到主螢幕的自訂資訊界面
* **三擊**：跳轉到主螢幕的自訂資訊界面
* **長按（小於 5 秒）**：切換副螢幕的下一個資訊界面
* **長按（大於 5 秒）**：進入設定選單

### 設定界面 <a href="#setting" id="setting"></a>

* **點選**：跳轉到下一設定項、或切換設定值
* **雙擊**：跳轉到下下一設定項
* **三擊**：跳轉到下下下一設定項
* **長按**：進入或退出某一設定項
